    SAME AS DELIVERY

In the order page, I have implemented the 'same as delivery address' checkbox in the order form. So in the preview form in the left, when we click the delivery radio button, the address field will become enabled. If we have fill in the address field, once we tick the same as delivery address, the street and postcode field for the billing address will automatically become invisible. And so, if we tried later in the form, the street and postcode field for the billing address will be filled with the delivery address. This is done by using java script.






    TYPES OF CREDIT CARDS

In the order page, I have implemented to choose either visa / mastercard / american express. When a card type has been chosen, and customer need to input card number based each requirement digit. Here in JavaScript, I have made an alert to print whether a card number is valid and has the requirement digit. 16 digits for both visa and mastercard and 18 digits for american express. (p.s. customer must choose online payment option in order card option and card details to be enabled)
